## Install
Put the file at the root of your user/root account:
* For Root : `/root/.vimrc`
* For User : `/home/user/.vimrc`
```shell
pspk@kali:~# ls -lhA | grep .vimrc
-rw-r--r-- 1 pspk pspk 3.4K Oct  3 00:44 .vimrc
```
