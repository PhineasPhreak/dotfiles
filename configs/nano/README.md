## Install

* Copy ".nanorc" to your home directory in `/home/$USER/.nanorc`

```shell
pspk@kali:~$ ls -lhA | grep .nanorc
```

## For help

Useful command for ignorercfiles
```shell
pspk@kali:~$ nano -I 'file-text'
```

Command for help:
```shell
pspk@kali:~$ nano --help
```
