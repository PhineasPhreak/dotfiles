## Package "rofi"

### Installation:
Update your Repositories:
```shell
$ sudo apt-get update
$ sudo apt-get install rofi
```

### Configuration Rofi:
Paste the 'config' file to the home directory
```shell
~/.config/rofi/config
```

### Shorcut:
Use the following command to lauch rofi:
```shell
rofi -show run
```
