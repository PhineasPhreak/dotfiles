## Installation :
* Create `~/.config/prompt/` and copy `prompt.bash`
* Edit the `.bashrc` file with :

```bash
# Use skeswa/prompt
if [ -f ~/.config/prompt/prompt.bash ]; then
    source ~/.config/prompt/prompt.bash
fi
```
