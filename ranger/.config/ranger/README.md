## Ranger

### Installation:
Install package:
```shell
$ sudo apt-get update
$ sudo apt-get install ranger
```
### Configuration:
* Copy all files:

`bookmarks`, `commands.py`, `commands_full.py`, `history`, `rc.conf`, `rifle.conf`, `scope.sh`, `tagged`

In `~/.config/ranger`

> **Note Important:** This configuration is the config by default of ranger
